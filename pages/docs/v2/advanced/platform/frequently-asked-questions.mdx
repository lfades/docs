## Missing Public Directory

This error occurs when your built project does not get output to the `/public` directory, preventing Now from uploading your built project.

To resolve this issue, provide your framework with instructions on outputting builds to a `/public` directory.

## Missing Build Script

This error occurs when you have a `package.json` file with no `build` script defined, preventing Now from building your project.

To resolve this issue, add a `build` script to your `package.json` file
